GET example

    http://sunshine.sinergis.it/maestro/ws/Action/light_control?PROCEDURE_ID=80097&DIMMERPERCENT=101&USERNAME=Sunshine

    Parameters
    • PROCEDURE_ID : id of the lamp which we would control
    • DIMMERPERCENT : percent of the lamp light
        • Values:
      • 0 - 100 -> range
      • 101 -> turn off the light
      • 102 -> turn on the light
      • 103 -> minimum consumption
    • USERNAME : Name of the actor doing the action
